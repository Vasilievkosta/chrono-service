# ChronoService Frontend

Клиентская часть сервиса по ремонту.  
Пользователь может оформить заказ, заполнив форму, и получить подтверждение на указанную почту.  

## 🚀 Технологии
- [Vite](https://vitejs.dev/) — сборка и разработка
- [React](https://react.dev/) — UI
- [TypeScript](https://www.typescriptlang.org/) — типизация
- [RTK Query](https://redux-toolkit.js.org/rtk-query/overview) — работа с API
- [Zod](https://zod.dev/) — валидация форм и данных

## 📌 Фукциональность
- Форма заказа с валидацией
- Выбор мастера в соответствующем городе на определенную дату и время  
- Отправка данных на сервер
- Подтверждение заказа по email
